Preguntas Introductoras

¿Qué es un framework?

Es una estructura previa en donde se puede realizar un proyecto

¿Qué es Bootstrap5 y cuál es su origen?

Bootstrap es un framework de código abierto para HTML, CSS Y JS, se utiliza para diseñar sitios web de una manera sencilla y rápida. Bootstrap, riginalmente llamado Blueprint fue desarrollado por Mark Otto y Jacob Thornton de Twitter, para fomentar la consistencia entre las herramientas internas.

¿Qué es diseño responsive en páginas web?

Es una técnica de diseño web que busca la visualización correcta de una página en distintos dispositivos.

¿Qué son las Github pages?

Permite subir repositorios de código para almacenarlo.Puedes colaborar en proyectos, y el sistema es código abierto , lo que significa que cualquiera persona puede modificar tu código, usarlo y mejorarlo.

¿Cuál es la diferencia entre elemento y componente web?

Los componentes de una página web son: imágenes, texto, audio, video, animaciones y otros contenidos y los elementos son los tipos de datos, bloques de código y otros.

¿Cuál es la importancia de documentar o comentar código?

Es importante documentar el código para explicar lo que se hace de manera que no solo la computadora entienda sino que además las personas vean el código y entiendan lo que se esta haciendo.

¿Por qué existe la sección de atribución en un sitio web?

Sirve para determinar qué contenido corresponde a qué parte de un esquema.

IntegraciÓn de Conocimientos

HTML+CSS

HTML y CSS son los principales lenguajes de programación web,y su uso principal es crear páginas y aplicaciones web. La diferencia crucial entre los dos es que el HTML se utiliza para la creación de las páginas web y el CSS se utiliza para controlar el estilo y el diseño de las páginas web.

Maquetado Responsive

El diseño responsive facilita la lectura y el acceso a los contenidos del portal o web, para hacer más fácil su uso.También facilita un acceso más rápido, ya que tardan menos en cargarse al estar optimizadas para estos móviles.

Github pages

es un sitio "social coding". Te permite subir repositorios de código para almacenarlo en el sistema de control de versiones

Bootstrap 5

Bootstrap es un framework CSS de código abierto que favorece el desarrollo web de un modo más sencillo y rápido. Incluye plantillas de diseño basadas en HTML y CSS con la que es posible modificar tipografías, formularios, botones, tablas, navegaciones, menús desplegables, etc.
